# Printing and Formal Documents

> Owner: Product · Last-updated: 2026-06-19 · Source-of-truth: docs/product/scope-register.md (scope), docs/plan/roadmap.md (sequence)

Support printable and PDF layouts for:

- application dossier
- evaluation summary
- committee decision
- mentorship report
- training transcript
- graduation report
- certificate
- sponsor report
- audit report

Requirements:

- Arabic and English
- RTL and LTR
- print-safe layout
- versioned templates
- signatures and verification links
- document numbering
- watermarking where required
