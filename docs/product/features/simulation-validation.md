# Program Simulation and Validation

> Owner: Product · Last-updated: 2026-06-19 · Source-of-truth: docs/product/scope-register.md (scope), docs/plan/roadmap.md (sequence)

## Configuration Validation

Before publication, validate:

- missing references
- unpublished dependencies
- invalid weights
- deadlocked stage paths
- missing graduation dependencies
- invalid role requirements
- unreachable workflow states
- missing notification templates

## Simulation

Allow program managers to simulate:

- applicant flow
- eligibility rules
- stage transitions
- score calculations
- workflow routing
- notifications
- graduation criteria

Simulation must not modify production records.
