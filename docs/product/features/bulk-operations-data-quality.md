# Bulk Operations, Data Quality, and Version Migration

> Owner: Product · Last-updated: 2026-06-19 · Source-of-truth: docs/product/scope-register.md (scope), docs/plan/roadmap.md (sequence)

## Bulk Operations

Support:

- invite
- evaluator assignment
- stage movement
- notifications
- exports
- document requests
- role assignment
- status updates

Each bulk operation requires:

- preview
- validation
- background execution
- partial-failure handling
- audit
- safe undo where possible

- invalid documents

- data-quality dashboards
- repair workflows
