# Onboarding and Progressive Disclosure

## User Onboarding

On first login:

1. confirm general profile
2. review role profiles
3. select organization or program
4. review required actions
5. explain profile sharing
6. show current program status

## Program Manager Onboarding

Use a guided setup:

1. Basic Information
2. Target Participants
3. Program Journey
4. Application Requirements
5. Evaluation Model
6. Delivery Activities
7. Graduation Rules
8. Review and Publish

## Progressive Disclosure

Advanced controls should remain collapsed by default:

- custom formulas
- parallel stages
- conditional routing
- escalation policies
- webhook actions
- advanced permission overrides

## Empty States

Every first-use screen must explain:

- what the feature does
- why it matters
- the first action
- required prerequisites
