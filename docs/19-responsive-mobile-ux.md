# Responsive and Mobile UX

## Desktop-First Tasks

- program configuration
- workflow design
- complex tables
- bulk assignment
- reporting
- advanced filtering

## Mobile-First Tasks

- notifications
- required actions
- session confirmation
- attendance
- task completion
- document upload
- quick feedback
- status review

## Requirements

- do not merely shrink desktop
- use adaptive navigation
- preserve essential actions
- avoid horizontal overflow
- provide mobile file upload
- support touch target sizing
- support portrait orientation
- test Arabic RTL on mobile
