# Usability Testing and Analytics

> Owner: UX · Last-updated: 2026-06-19 · Source-of-truth: docs/product/scope-register.md (scope), docs/plan/roadmap.md (sequence)

## Required Role Tests

### Founder

- complete profile
- apply to a program
- upload documents
- understand next action
- confirm a session

### Mentor

- review startup context
- accept assignment
- schedule session
- record notes
- create tasks

### Evaluator

- review application
- understand rubric
- save draft
- submit score
- declare conflict

### Program Manager

- create program
- configure stages
- build application
- assign evaluators
- identify delayed participants
- approve graduation

## Acceptance Targets

- task success at least 90% for critical flows
- no critical usability errors
- no inaccessible blocking controls
- no repeated entry of reusable profile data
- essential mobile tasks fully completable
- Arabic and English validated

## Analytics Events

- application_started
- application_section_completed
- validation_error
- application_abandoned
- application_submitted
- document_upload_failed
- mentor_assignment_accepted
- evaluation_started
- evaluation_submitted
- training_completed
- graduation_approved
- search_no_results
- support_requested

## Privacy

Analytics must not capture sensitive field values.
