# Testing Strategy

> Owner: Quality · Last-updated: 2026-06-19 · Source-of-truth: docs/product/scope-register.md (scope), docs/plan/roadmap.md (sequence)

Required: unit, feature, API, authorization, tenant-isolation, contract, integration, end-to-end, security, performance, and accessibility tests.
