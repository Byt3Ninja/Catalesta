# Release Gates

> Owner: Delivery · Last-updated: 2026-06-19 · Source-of-truth: docs/product/scope-register.md (scope), docs/plan/roadmap.md (sequence)

## Gate 1: Foundation

Required:

- repository and CI
- mock OIDC
- identity
- tenancy
- RBAC
- audit baseline

## Gate 2: Configurable Program

Required:

- programs
- cohorts
- stages
- forms
- workflows
- publication

## Gate 3: Application and Selection

Required:

- applications
- snapshots
- documents
- eligibility
- assessments
- decisions

## Gate 4: Program Delivery

Required:

- role assignments
- tasks
- mentorship
- training
- integrations

## Gate 5: Graduation and Reporting

Required:

- final evaluation
- graduation
- certificates
- alumni
- reporting
- exports

## Gate 6: Production Readiness

Required:

- security hardening
- observability
- performance
- backup restore
- disaster recovery
- accessibility
- UAT
- cutover plan
