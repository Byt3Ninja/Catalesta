# Product Scope

## Product Purpose

The platform enables organizations to create and operate configurable incubation and acceleration programs.

Each program may define:

- Application stages
- Eligibility checks
- Initial screening
- Interviews
- Mentorship
- Training
- Assignments
- Milestone reviews
- Final evaluations
- Demo days
- Graduation
- Alumni follow-up

## Core Actors

- Platform Administrator
- Organization Administrator
- Program Director
- Program Manager
- Program Coordinator
- Founder
- Co-Founder
- Startup Team Member
- Mentor
- Trainer
- Evaluator
- Judge
- Service Provider
- Investor
- External Observer

## Core Capabilities

### Organization Management

- Multi-tenant organization setup
- Branding
- Users and permissions
- Program templates
- Shared role pools
- Organization-level configuration

### Program Management

- Program creation
- Cohort management
- Program cloning
- Configurable stages
- Stage sequencing
- Conditional branching
- Parallel stages
- Timelines
- Policies
- Program-specific roles

### Application Management

- Dynamic application forms
- Reusable Startup Gate profile data
- Conditional questions
- Required documents
- Eligibility rules
- Draft and submission lifecycle
- Application review
- Return-for-update workflow

### Assessment

- Weighted criteria
- Categories and subcategories
- Rubrics
- Disqualifying criteria
- Multiple evaluators
- Blind and double-blind review
- Committee decisions
- Final score calculation

### Mentorship

- Mentor directory integration
- Mentor matching
- Capacity and availability checks
- Conflict-of-interest checks
- Mentorship plans
- Session scheduling
- Tasks and outcomes
- Mutual feedback

### Training

- Courses
- Modules
- Sessions
- Attendance
- Assignments
- Quizzes
- Surveys
- Results
- Certificates

### Graduation

- Graduation criteria
- Final committee
- Certificates
- Alumni status
- Follow-up plans
- Outcome publication back to Startup Gate

## Out of Scope for Initial Release

- Full investor marketplace
- Billing and subscription management
- Advanced AI scoring
- Predictive analytics
- Full BPMN execution engine
- Cross-region active-active deployment
- Multi-database tenancy
