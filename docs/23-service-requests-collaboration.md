# Service Requests and Collaboration

## Service Requests

Lifecycle:

```text
Startup requests service
→ Eligibility
→ Provider matching
→ Quotation or scope
→ Approval
→ Delivery
→ Evidence
→ Acceptance
→ Rating
→ Closure
```
