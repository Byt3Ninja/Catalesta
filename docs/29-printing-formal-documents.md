# Printing and Formal Documents

Support printable and PDF layouts for:

- application dossier
- evaluation summary
- committee decision
- mentorship report
- training transcript
- graduation report
- certificate
- sponsor report
- audit report

Requirements:

- Arabic and English
- RTL and LTR
- print-safe layout
- versioned templates
- signatures and verification links
- document numbering
- watermarking where required
