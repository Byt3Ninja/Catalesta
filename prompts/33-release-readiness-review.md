# Claude Task: Release Readiness Review

## Goal

Perform a complete architecture, security, data, integration, and operations review before release.

## Review Areas

- module boundaries
- data ownership
- tenant isolation
- authorization
- shared contracts
- event compatibility
- migration safety
- rollback
- observability
- backup restore
- performance
- accessibility
- external integrations
- Startup Gate replacement readiness

## Output

Produce a release-readiness report with:

- blockers
- high-risk findings
- medium-risk findings
- accepted risks
- remediation tasks
- go/no-go recommendation
